# pdf-to-jpeg

このツールは、PDF ファイルをブラウザだけで安全かつ高品質な JPEG 画像に変換する無料の Web アプリです。  
インストール不要・アップロード不要・アカウント不要・スマホ対応で、誰でも簡単に使えます。

## 特徴

- ブラウザだけで動作（クライアントサイド処理）
- 複数ファイル・複数ページ対応
- 高画質 JPEG（品質約 92〜95%）
- ZIP 自動生成
- データはサーバーに送信されません
- アップロード不要（すべてブラウザ内で処理）

## 使い方

1. PDF を選択  
2. 変換が始まる
3.  ZIP を保存

詳細な使い方は [guide](https://jun-ju-waaaaa.github.io/pdf-to-jpeg/guide/) をご覧ください。

## プライバシーと安全性

このツールはすべての処理をブラウザ内で行い、PDF や画像データを外部サーバーに送信することはありません。  
詳細は [プライバシーポリシー](https://jun-ju-waaaaa.github.io/pdf-to-jpeg/privacy/) をご覧ください。
